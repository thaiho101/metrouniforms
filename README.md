# Uniform Voucher Management System

## 🚀 Overview
A full-stack internal system built to replace manual voucher tracking in warehouse operations, improving speed, accuracy, and reporting efficiency.

## 🔥 Key Features
- Search-based customer validation
- Real-time voucher lookup by name and organization
- CSV export for reporting
- Dashboard for tracking usage across organizations

## 📊 Impact
- Reduced lookup time by **~5x** (10–20s → 2–3s)
- Reduced reporting time from **15 minutes to under 1 minute**
- Eliminated risk of data loss from paper-based tracking

## 🛠 Tech Stack
- PHP
- MySQL
- JavaScript
- HTML/CSS
- Google Charts
